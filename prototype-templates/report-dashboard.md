# Report Dashboard Template

Purpose: statistics, trends, analysis, and operational tracking.

Use when the PRD mentions report, statistics, trend, chart, KPI, or dashboard.

Required structure:
- KPI summary cards.
- Table or chart-like blocks using CSS, no external libraries.
- Filter controls for time, department, status, or category.
- Ranking, trend, or breakdown area.

Interaction requirements:
- Filters should be clickable or visibly interactive.
- KPI cards can navigate to filtered list.
