# Responsive Rules

- Desktop can use side navigation plus main workspace.
- At widths below 920px, grids, form rows, detail grids, and side layouts should stack into one column.
- Buttons and inputs must not overflow.
- Tables may remain scrollable or simplify density, but the page must remain usable.
- Do not scale font size with viewport width.
- Use stable dimensions for navigation, toolbar, cards, and action buttons.
