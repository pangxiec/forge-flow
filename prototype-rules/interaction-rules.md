# Interaction Rules

- Main navigation must switch pages locally.
- List rows or view buttons must enter detail pages.
- Create/edit submit must navigate to detail or next workflow state.
- Approval pages must include clickable approve and reject actions when workflow is present.
- Detail pages must provide a back action.
- Tabs, filters, row selection, and primary actions should have local JavaScript behavior or visible feedback.
- Do not request backend APIs.
